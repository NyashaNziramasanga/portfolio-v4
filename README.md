# Portfolio v4

A simple portfolio layout with a left navigation panel and scrollable content sections, built with React, TypeScript, and shadcn/ui.

## Run locally

```bash
yarn install
yarn dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
yarn build
yarn preview
```
